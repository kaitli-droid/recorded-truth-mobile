# Recorded Truth (Mobile)

Voice-driven therapeutic journaling application built with Flutter.

## Overview

Recorded Truth allows users to capture voice journals and convert them into structured entries using speech-to-text and NLP. The app is designed for emotional accessibility, reliability, and privacy.

This repository contains the mobile client.

## Key Focus Areas

- Audio capture and processing
- Speech-to-text integration
- Clean, predictable state management
- Accessibility-first UI design
- Secure handling of sensitive user data

## Tech Stack

- Flutter (Dart)
- Firebase (Auth, Firestore, Cloud Functions)
- Speech-to-Text APIs
- NLP
- REST APIs

## Architecture Notes

The app follows a layered architecture with clear separation between:
- UI
- State management
- Domain logic
- Infrastructure and services

This keeps features isolated, testable, and easier to evolve.
